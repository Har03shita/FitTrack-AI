# FitTrack-AI – Fitness Microservices Backend

A microservices-based fitness tracking backend built using Spring Boot.

## Tech Stack
- Spring Boot
- RabbitMQ
- Eureka Service Discovery
- Docker
- MySQL + MongoDB

## Services
- Activity Service
- User Service
- AI Insights Service
- API Gateway
- Config Server
- Eureka Server

## Features
- RESTful APIs
- Async messaging with RabbitMQ
- Scalable containerized deployment
